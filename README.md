# 2DR-DiceBot
## Environment
Discord
## Description
Roll the dice by 2DR.  
Critical if boxcars.Fumble when the dice match five.  
サイコロの2DRを振ることのできるアプリです。   
低い出目の方が良い出目です。   
また、6が出ると特殊なポイントを手に入れます。   
6ゾロがクリティカル,5ゾロがファンブルと表示されます。   
通常のダイスも振れます。
